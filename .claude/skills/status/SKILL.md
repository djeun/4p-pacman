PLAN.md와 PROGRESS.md를 읽고 아래 형식으로 현재 상태를 보고하라.

1. **PROGRESS.md** 를 읽어 완료된 작업과 현재 진행 중인 작업을 파악한다.
2. **PLAN.md** 를 읽어 남은 작업 목록과 현재 Phase를 파악한다.

보고 형식:
- 현재 Phase와 전체 진행률 (완료된 체크박스 / 전체 체크박스)
- 완료된 작업 요약 (최근 3~5개)
- 현재 진행 중인 작업 (있으면)
- 다음으로 해야 할 작업 (PLAN.md 기준 미완료 항목 중 최우선 3개)
- 막혀있거나 주의할 사항 (PROGRESS.md 메모 참고)
